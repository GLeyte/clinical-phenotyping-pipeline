# SHADE: Deep Density-based Clustering

This folder contains the official Python implementation for the paper "SHADE: Deep Density-based Clustering" from the following paper:

Beer, Anna, Pascal Weber, Lukas Miklautz, Collin Leiber, Walid Durani, Christian Böhm, and Claudia Plant. "SHADE: Deep density-based clustering." *Proceedings of the 28th ACM SIGKDD conference on knowledge discovery and data mining*, 2024. arXiv preprint [arXiv:2410.06265](https://arxiv.org/abs/2410.06265), 2024. https://doi.org/10.48550/arXiv.2410.06265
