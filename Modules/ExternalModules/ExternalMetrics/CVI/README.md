# Python CVI Implementations

This folder contains a selection of Python implementations of Cluster Validity Index (CVI) metrics.

These implementations are based on the [ASCVI library](https://github.com/g-schlake/ASCVI) from the following paper:

Schlake, Georg Stefan, and Christian Beecks. "Validating Arbitrary Shaped Clusters-A Survey." *2024 IEEE 11th International Conference on Data Science and Advanced Analytics (DSAA)*. IEEE, 2024.
[https://doi.org/10.1109/DSAA61799.2024.10722773](https://doi.org/10.1109/DSAA61799.2024.10722773)

Citations for all used CVIs can be found in the docstrings of their respective classes.
