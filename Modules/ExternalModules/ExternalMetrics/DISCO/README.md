# DISCO: Internal Evaluation of Density-Based Clustering

This folder contains the official Python implementation for the paper "DISCO: Internal Evaluation of Density-Based Clustering" from the following paper:

Beer, Anna, Lena Krieger, Pascal Weber, Martin Ritzert, Ira Assent, and Claudia Plant. "DISCO: Internal Evaluation of Density-Based Clustering." arXiv preprint arXiv:2503.00127, 2025. https://doi.org/10.48550/arXiv.2503.00127
